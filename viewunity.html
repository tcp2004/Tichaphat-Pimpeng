Goodreads Redesign

Project Duration: June 2023 - October 2023

Overview

This project focused on enhancing the user experience (UX) and user interface (UI) of Goodreads, targeting improvements in usability, aesthetics, and functionality.

Challenges & Goals

  • Simplify navigation for users to discover and track books more easily.
  • Improve the visual design to align with modern UX/UI trends.
  • Integrate features like personalized recommendations and streamlined review workflows.

Outcome Highlights

  • Redesigned homepage for a clean, user-friendly layout.
  • Revamped search and filter tools with intuitive interactions.
  • Enhanced mobile experience to ensure seamless cross-device usage.
